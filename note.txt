Pour utiliser Functions, vous devez installer les outils de ligne de commande Firebase à l'aide de npm

$ npm install -g firebase-tools

Si vous avez déjà installé les outils de ligne de commande Firebase, exécutez de nouveau la commande d'installation pour vous assurer que vous disposez bien de la dernière version.

Ouvrez une fenêtre de terminal et accédez au répertoire de votre code :

Lancez votre projet : $ firebase init

Déployez vos fonctions : $ firebase deploy
